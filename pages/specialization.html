Mobile Application Testing
Training Plan
Specialist 1

Introduction

Mobile Application Testing involves evaluating the software against its technical and business requirements, in order to determine its effectiveness. Mobile Application Testing likewise involves implementing techniques in order to uncover software defects

With these being said, You_Source is committed to provide our Quality Assurance Engineers with a Mobile Application Testing course content guide to make their learning experience as simple as possible. The details of the Mobile Application Testing course content guide can be found in the sections below.

The QA Mobile Specialist

Mobile Application Tester is focused on checking mobile application software for functionality, usability, and consistency on handheld mobile devices. Mobile Application Testing can either be automated or manual.

Course levels

QA Division identified 2 Levels of Mobile Application Testing training to become You-Source QA Mobile Specialist.

Expectation:

As a QA mobile specialist you are expected to be able to:

  • To Identify issues and defects specifically on mobile applications, mobile web applications and hybrid applications across different mobile OS and devices.
  • To create a testing plan and strategize testing using the different mobile testing type and techniques.
  • To start the testing activities of a mobile project

Requirement:

  • In order to qualify for mobile specialist training you need to finish the Mobile testing introduction to acquire basic understanding on how mobile applications differ from web based applications. This document provides reading materials and video materials to help you go through the course of training.
  • A QA who would like to take the course should be atleast Quality Analyst 2 to qualify.
